# ⚡ AorusGrants

> Modern, GUI-based rank management plugin powered by LuckPerms  
> Designed for fast, intuitive, and fully configurable staff workflows.

---

## ✨ Features

- 🧩 **5-step GUI flow** for granting and demoting ranks
- 🔒 **100% LuckPerms API integration** — no custom permission system
- ⏳ **Temporary grants** with day/week/month accumulators
- ⚙️ **Fully configurable menus** via `menus.yml`
- 📊 **Session manager** with automatic expiration
- ⚡ **Tab-complete support** for `/ag <player>`
- 🎨 **Custom prefixes, materials, and layouts**

## 🚀 Installation

1. Download the latest `.jar`
2. Place it in your `plugins/` folder
3. Install **LuckPerms**
4. Start or restart your server
5. Configure:
    - `plugins/AorusGrants/config.yml`
    - `plugins/AorusGrants/menus.yml`
